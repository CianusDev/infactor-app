Billing App (nom temporaire) est un micro-saas (MVP) destiné aux freelances et petites entreprises pour creer et personnaliser et imprimer des factures professionnelles rapidement. L'application offre une interface conviviale permettant aux utilisateurs de générer des factures en quelques clics, avec des options de personnalisation telles que l'ajout de logos, la modification des couleurs et la sélection de modèles prédéfinis.

Fonctionnalités principales :
1. Création de factures : Génération rapide de factures professionnelles avec des champs personnalisables (nom du client, description des services, montants, taxes, etc.).
2. Personnalisation : Options pour ajouter des logos, choisir des couleurs et sélectionner parmi plusieurs modèles de factures.
3. Gestion des clients : Base de données simple pour stocker les informations des clients et faciliter la création de factures récurrentes.
4. Exportation et impression : Possibilité d'exporter les factures au format PDF directement depuis l'application.

Technologies utilisées :
- Frontend : Next.js 16 pour une interface utilisateur réactive et moderne.
- Backend : Next.js 16 API routes pour gérer la logique serveur et les opérations CRUD.
- Base de données : Posgre SQL pour stocker les informations des clients et des factures.
- Authentification : JWT et GOOGLE OAuth pour sécuriser l'accès à l'application.
- ORM : Prisma pour interagir avec la base de données de manière efficace.
- Hébergement : Vercel pour le déploiement facile et rapide de l'application.
Objectif :
L'objectif principal de ce micro-saas est de fournir aux freelances et petites entreprises un outil simple et efficace pour gérer leurs factures, leur permettant ainsi de se concentrer davantage sur leur activité principale plutôt que sur les tâches administratives.
